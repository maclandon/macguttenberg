MACLANDON.COM — DESIGN RULES
=================================

COLORS
------
Background primary: #E8E4DC (--bg)
Background secondary: #DDD9D0 (--bg2)
White: #FAFAF7 (--white)
Card background: #F5F2EC (--card)
Border: #D0CAC0 (--border)
Text primary: #1C1A16 (--text)
Text mid: #48443C (--text-mid)
Text muted: #7A7468 (--text-muted)
Accent blue: #7fa0c3 (--accent)
Accent blue dark: #3D6B9A (--accent2)
Dark background: #1C1A16 (--dark)
Dark background 2: #2A2720 (--dark2)

TYPOGRAPHY
----------
Display/heading font: Fraunces (serif) — used for all h1, h2, h3, section headlines
Body font: DM Sans (sans-serif) — used for all body text, labels, nav, buttons
h1: clamp(2rem, 4vw, 4rem), weight 900, line-height 0.98, letter-spacing -0.03em
h2 (sec-h2): clamp(1.8rem, 2.8vw, 2.6rem), weight 700, line-height 1.1, letter-spacing -0.025em
Body text: 0.95–0.97rem, weight 300, line-height 1.75–1.9
Small labels/tags: 0.68rem, weight 500, letter-spacing 0.12em, uppercase
Italic accent text in headings: animated blue gradient (#7fa0c3 → #3D6B9A → #a0c4e8)

BUTTONS
-------
Primary (btn-dark): background var(--dark), white text, font-size 0.88rem, weight 500, padding 0.8rem 1.6rem, border-radius 6px
Accent (btn-accent): background var(--accent), white text, same sizing as btn-dark, border-radius 6px
Outline (btn-price-outline): transparent background, accent color text, 1px solid border, border-radius 6px
Ghost/soft (btn-soft): no background, muted text, underline only
Nav CTA: 0.72rem, weight 600, uppercase, white text, 1px white border, padding 0.45rem 1.1rem, border-radius 4px
All buttons: scale(0.96) on active/press
Mobile: all primary buttons full-width, min-height 48px

CARDS
-----
Hero card: background var(--card), border-radius 24px, padding 3rem, 1px border var(--border), box-shadow 0 2px 40px rgba(28,26,22,.07)
Section cards (s-card, p-card): background var(--card), border-radius 16px, padding 2.25rem, 1px border
Pricing cards: background var(--card), border-radius 18px, padding 2rem, 1px border
Popular pricing card: background var(--dark), no border
Right-list items: background var(--card), border-radius 10px, padding 0.85rem 1rem, 1px border
Result items (case study): background rgba(255,255,255,.05), border-radius 14px, padding 1.25rem 1.5rem
Testimonial cards: background var(--card), border var(--border)
CTA section card: background var(--dark), border-radius 20px, padding 2.5rem

NAVIGATION
----------
Height: 56px, fixed top
Background: rgba(28,26,22,0.92) with backdrop-filter blur(20px)
Bottom border: 1px solid rgba(255,255,255,.06)
Nav links: 0.72rem, weight 500, uppercase, letter-spacing 0.1em
Active link: white underline 2px, slides in on hover
Desktop links: About, Services, Pricing, Process, Contact, Blog
Mobile: hamburger icon (≡ / ✕), hides nav links and social icons
Mobile nav hides: .nav-links, .nav-li (social icons), .nav-cta (Book a Call button)

SPACING
-------
Section padding desktop: 5.5rem 4rem
Section padding mobile: 3rem 1.5rem
Max content width: 1100px, centered
Grid gaps: 2–4rem desktop, 1–1.5rem mobile

IMAGES
------
Hero photo desktop: cutout PNG, bottom-aligned, drop-shadow
Hero photo mobile: circle avatar shown inside card next to value prop
About image: full bleed, min-height 520px, hover zoom scale(1.05)
Case study images: 2-col grid, 280px tall desktop, 140px tall mobile
Avatars (testimonials): 52px circle, 2px border var(--border)

ANIMATIONS
----------
Page load: pagefade opacity 0→1, 0.6s
Hero elements: staggered translateY(20px)→0 with opacity, delays 0.1–0.52s
Gradient text: background-position animates 0%→100%→0%, 4s infinite
Hero card: float animation, -8px to 0, 6s infinite (disabled on mobile)
Blob background: morphing border-radius, 8s infinite
Scroll reveal: .reveal class, opacity 0 + translateY(24px) → visible on scroll
Back to top button: circular progress ring tied to scroll position
Preloader: logo scales 1.9→1 with opacity, 0.5s delay

MOBILE BREAKPOINT (max-width: 900px)
--------------------------------------
Hero: single column, height auto, card animation disabled
h1: 1.75rem fixed
Hero card padding: 2rem 1.75rem 1.75rem
Primary CTA button: full-width, justify-content center
Hero sub-copy (.hero-sub): hidden
Social icons in nav: hidden
Book a Call in nav: hidden
Hero right column (photo): hidden by default
Stat bar: 3-column horizontal grid, compressed font sizes
Case study 2x2 metrics grid: grid-template-columns 1fr 1fr
Pricing: horizontal swipe carousel, 78vw card width, scroll-snap, dot indicators
Process steps: vertical timeline with accent-colored numbered circles
Forms: min-height 48px inputs, font-size 16px (prevents iOS zoom)
Sticky CTA bar: fixed bottom, full-width Book a Call button

SECTION TAGS
------------
All section eyebrows: 0.68rem, weight 500, uppercase, letter-spacing 0.12em, accent color
Tag line before: 14px wide, 1.5px tall accent-colored bar

SOCIAL PROOF
------------
Testimonial quotes: italic em tags for key phrases
Testimonial names: First name + last initial only (e.g. "Mark G.")
Long testimonials: truncated to 3 lines on mobile with "Read more" toggle
Star ratings / metrics: Fraunces serif, large, accent color numbers

EASTER EGG
----------
Double-click logo triggers Matrix rain animation using "MACLANDON" + Japanese characters in accent blue, 180 frames, fades out
