Tone: warm, soft, calming — never clinical or cold

### Key CSS classes
.btn-sage — primary green button (rounded pill)
.btn-ghost — outlined secondary button
.section-tag — small uppercase label above headings
.section-heading — serif heading with optional <em> italic
.reveal / .reveal-delay-1/2/3 — scroll animation classes
.cred-pill — credential badge (PT, DPT, GCS, 200RYT)
.service-card — service block with number
.condition-item — what I treat list item with dot
.blog-card — blog preview card
.testimonial-card — patient review card
.faq-question / .faq-answer — accordion FAQ (JS toggleFaq())
.hero-photo — full bleed headshot image
.contact-form — Formspree form

## Writing style
- First person (I, my, me) — Marissa's voice
- Warm, empowering, never preachy
- Sentence case always — never ALL CAPS headings
- Short paragraphs, plain language
- Avoid overly clinical jargon

## Deployment
- GitHub repo: maclandon/marissa-joy-wellness
- Hosted on Cloudflare Pages — auto-deploys on every push to main
- Domain: marissajoywellness.com

## Important notes
- Always keep headshot.jpg filename exactly as is
- Blog posts link back to index.html#blog
- Nav links use anchor tags (#services, #about, #blog, #faq, #contact)
- Mobile hamburger menu controlled by toggleMenu() JS function
- FAQ accordion controlled by toggleFaq() JS function
- Never remove Formspree action URL from the contact form
- Privacy policy links to privacy-policy.html from footer
